## 一、项目概述

一个面向 C 端用户的影视内容平台，**YouTube 暗色主题风格**，支持电影浏览/搜索/收藏、AI 智能客服对话、用户会员体系、内容评论等完整业务闭环。

| 指标 | 数据 |
|------|------|
| 总提交数 | 25 |
| 前端源码文件 | 63 |
| 后端源码文件 | 40 |
| 总代码行数 | ~5,700 |
| 数据库电影数 | 100（豆瓣 Top 250） |
| 支持角色 | USER / ADMIN |
| 支持设备 | 手机 / 平板 / 桌面 五档断点 |

## 六、API 端点总览

### 认证 `/api/auth`

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| POST | `/api/auth/register` | 注册 | Public |
| POST | `/api/auth/login` | 登录 -> JWT | Public |
| GET | `/api/auth/me` | 当前用户信息 | Login |

### 电影 `/api/movies`

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | `/api/movies` | 列表 (page/genre/sort/q) | Public |
| GET | `/api/movies/{id}` | 详情 | Public |
| GET | `/api/movies/genres` | 分类列表 | Public |

### 海报 `/api/posters`

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | `/api/posters/{movieId}` | 二进制图片 (image/webp) | Public |

### 评论 `/api/comments`

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | `/api/comments?movieId=X` | 评论列表 | Public |
| POST | `/api/comments` | 发表评论 | Login |
| DELETE | `/api/comments/{id}` | 删除评论 | Owner/ADMIN |

### 片单 `/api/watchlist`

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET/POST | `/api/watchlist` | 获取/添加 | Login |
| DELETE | `/api/watchlist/{movieId}` | 移除 | Login |

### AI `/api/ai`

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| POST | `/api/ai/chat` | 对话 (SSE) | Public |

### 管理 `/api/admin`

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | `/api/admin/stats` | 统计 | ADMIN |
| GET | `/api/admin/videos/pending` | 待审核 | ADMIN |
| PUT | `/api/admin/videos/{id}/approve` | 通过 | ADMIN |
| PUT | `/api/admin/videos/{id}/reject` | 拒绝 | ADMIN |

## 七、数据库表

## 九、启动命令

### Quick Start

```
Double-click start-all.bat
```

### Manual Start

```bash
# 1. Docker
docker compose up -d

# 2. Backend (new terminal)
set JAVA_HOME=D:\JDK21
set PATH=%JAVA_HOME%\bin;D:\maven\maven-mvnd-2.0.0-rc-3-windows-amd64\mvn\bin;%PATH%
cd backend
mvn spring-boot:run "-Dspring-boot.run.jvmArguments=-DDEEPSEEK_API_KEY=your-key"

# 3. Frontend (new terminal)
cd frontend
npm install   # first time only
npm run dev
```

## 一、设计参考：YouTube 布局体系

### 1.1 对标说明

本项目的整体布局体系以 **YouTube（youtube.com）** 为唯一设计参考基准。所有页面继承 YouTube 的核心交互范式：

| YouTube 元素 | 影视平台适配 |
|-------------|------------|
| 视频缩略图 (16:9) | 电影海报卡片 (2:3) |
| 频道头像 → 频道名 | 评分徽章 → 导演/类型文字 |
| 观看量 · 发布时间 | ⭐评分 · 上映年份 |
| 左侧导航分类 | 电影类型分类（剧情/科幻/爱情...） |
| 顶部 Category Chips | 类型快速筛选横滚条 |
| 搜索框（居中圆形） | 电影搜索（居中圆形） |
| 视频详情页（播放器+右侧推荐） | 电影详情页（海报大图+下方推荐网格） |
| 深色主题 `#0F0F0F` | 同一深色背景体系 |

## 二、设计系统 (Design System)

### 2.1 色彩方案 —— YouTube Dark Theme

> 直接参考 YouTube 深色模式色板，适配影视平台调性

| 色阶 | CSS 变量名 | 色值 | 用途 |
|------|-----------|------|------|
| **页面背景** | `--bg-primary` | `#0F0F0F` | 页面主背景，Sidebar 背景 |
| 卡片/悬浮背景 | `--bg-secondary` | `#212121` | 卡片 Hover、搜索框、下拉菜单背景 |
| 输入框背景 | `--bg-tertiary` | `#303030` | Chip 默认态、输入框、分隔线 |
| **主文字** | `--text-primary` | `#FFFFFF` | 标题、正文、导航标签 |
| 次级文字 | `--text-secondary` | `#AAAAAA` | 描述文字、频道名、元信息 |
| 禁用文字 | `--text-disabled` | `#717171` | 占位符、禁用态文字 |
| **品牌色** | `--brand-primary` | `#6366F1` | 主按钮、选中态、AI 元素（替代 YouTube 红色） |
| 品牌色深 | `--brand-hover` | `#4F46E5` | 按钮 Hover |
| **评分色** | `--rating-star` | `#F59E0B` | 星级评分、收藏激活态 |
| 链接色 | `--link-color` | `#3B82F6` | 超链接 |
| 成功 | `--success` | `#10B981` | 审核通过 |
| 危险 | `--danger` | `#EF4444` | 审核拒绝、错误 |
| 卡片边框 | `--border-subtle` | `rgba(255,255,255,0.08)` | 卡片分割（极淡） |

> **原则**：YouTube 不用明显的卡片边框，内容区块之间靠**间距和背景微差**自然分离。本项目遵循同样原则。

### 2.2 字体系统 —— YouTube Style

| 层级 | TailwindCSS | 字重 | 使用场景 |
|------|------------|------|---------|
| 页面标题 | `text-2xl` (24px) | Bold 700 | Sidebar 分类标题、页面 H1 |
| 卡片主标题 | `text-sm` (14px) / `text-base` (16px) | Medium 500 | **电影名称**（14px 网格/16px 列表），最多 2 行截断 `line-clamp-2` |
| 二级信息 | `text-xs` (12px) | Regular 400 | 导演、年份、评分数字、观看量（`#AAAAAA`） |
| Chip 标签 | `text-sm` (14px) | Medium 500 | 类型筛选 Chip |
| 侧边栏项 | `text-sm` (14px) | Regular 400 | Sidebar 导航项 |
| 正文 | `text-sm` (14px) | Regular 400 | 描述、评论内容，行高 `leading-5` |

**字体栈**：`"Roboto", "Arial", "PingFang SC", "Microsoft YaHei", sans-serif`

> YouTube 本尊使用 Roboto。中文环境下自动降级到 PingFang SC / Microsoft YaHei。

### 2.3 间距与尺寸

| 场景 | 值 | TailwindCSS |
|------|-----|------------|
| TopBar 高度 | 56px | `h-14` |
| Sidebar 展开宽度 | 240px | `w-60` |
| Sidebar 折叠宽度 | 64px | `w-16` |
| 网格列间距 | 16px | `gap-4` |
| 网格行间距 | 40px | `gap-y-10` |
| CategoryChips 间距 | 12px | `gap-3` |
| 卡片内部文字间距 | 12px（缩略图底到标题） | `mt-3` |
| 页面内容区左右边距 | 24px | `px-6` |
| 卡片圆角 | 12px | `rounded-xl` |
| 按钮/Chip/搜索框圆角 | 9999px（全圆） | `rounded-full` |
| 头像/Logo 圆角 | 50%（正圆） | `rounded-full` |

### 2.4 动效规范 —— YouTube 式微交互

| 交互 | 效果 |
|------|------|
| 卡片 Hover | **无缩放**，仅缩略图亮度轻微变暗 + 文字标题变为品牌色（YouTube 风格） |
| 卡片缩略图 Hover | `brightness-75 transition-all duration-200`，不缩放 |
| Sidebar 项 Hover | `bg-[#272727] rounded-lg` |
| Chip 选中态 | 白色背景 + 黑色文字（`bg-white text-black`），非选中 `bg-[#303030] text-white hover:bg-[#3F3F3F]` |
| 搜索框 Focus | 边框从透明变为 `border-brand-primary`，宽度从默认 540px 轻微加宽 |
| Sidebar 折叠/展开 | `transition-[width] duration-200 ease-in-out` |
| 下拉菜单 | `animate-in fade-in slide-in-from-top-2 duration-150` |

### 2.5 图标

使用 **Lucide Icons**（与 YouTube 使用的 Material Icons 风格接近），统一 `size={24}` `strokeWidth={1.5}`（比此前更轻量，更像 YouTube）。

## 三、全局组件详细设计

### 3.1 TopBar（固定顶部导航栏）

```
背景：bg-[#0F0F0F]（与页面背景同色，靠 z-50 浮于内容之上）
高度：h-14 (56px)
布局：grid grid-cols-[auto_1fr_auto] items-center px-4 gap-4

┌──────────────────────────────────────────────────────────────┐
│ [☰]  [🎬 电影APP]     [🔍━━━━━━━━━━━━━━━] [🎤]   [🔔] [👤]   │
│  左区                   中区（弹性居中）              右区     │
└──────────────────────────────────────────────────────────────┘

左区 (flex, gap-2):
  - ☰ 汉堡按钮：w-10 h-10 rounded-full hover:bg-[#272727] flex items-center justify-center
    点击 → Sidebar 折叠/展开
  - Logo：text-xl font-bold（不使用图片，纯文字 🎬+品牌名），点击跳首页

中区 (flex-1, max-w-[640px], mx-auto):
  - 搜索框容器：flex-1 relative
  - <input>：w-full h-10 rounded-full bg-[#121212] border border-[#303030] 
    focus:border-brand-primary focus:bg-[#0F0F0F] px-4 pl-10 text-white 
    placeholder:text-[#888]
  - 搜索图标在左侧：absolute left-3 top-1/2 -translate-y-1/2 text-[#888] w-5 h-5
  - 🎤 语音按钮（可选装饰）：w-10 h-10 rounded-full bg-[#272727] hover:bg-[#3F3F3F]
    flex items-center justify-center ml-2

右区 (flex items-center gap-2):
  - 🔔 通知铃铛：w-10 h-10 rounded-full hover:bg-[#272727]，有未读时右上角小红点
  - 👤 用户头像：w-8 h-8 rounded-full（登录后显示，未登录显示 [登录] 文字按钮）
    点击 → 下拉菜单（bg-[#212121] rounded-xl border border-[#303030] shadow-2xl）
```

**TopBar 关键行为**：
- 始终固定在页面顶部，不随滚动消失
- 向下滚动时，搜索框获得极淡的底部阴影（`shadow-[0_1px_3px_rgba(0,0,0,0.5)]`）
- 移动端：中区搜索框收缩为仅搜索图标，点击后展开为全宽搜索

**Sidebar 导航内容**：

### 3.3 CategoryChips（类型快速筛选条）

```
位置：TopBar 下方，主内容区上方（紧随 TopBar）

└───────────────────────┘
```

**响应式列数**：
```
< 640px  (sm) : 2 列（手机横屏）
< 1024px (md) : 3 列（平板竖屏）
< 1280px (lg) : 4 列（小桌面）
< 1536px (xl) : 5 列（大桌面 / YouTube 默认）
≥ 1536px (2xl): 6 列（超大屏）
```

**滚动行为**：使用 IntersectionObserver 监听最后一个卡片，触底时自动 `GET /api/movies?page=N&pageSize=24` 追加数据。

```javascript
// lib/api.js
const API_BASE = process.env.NEXT_PUBLIC_API_BASE_URL || 'http://localhost:8080';

export async function apiFetch(url, options = {}) {
  const token = typeof window !== 'undefined' 
    ? localStorage.getItem('token') 
    : null;

  const headers = {
    'Content-Type': 'application/json',
    ...(token && { Authorization: `Bearer ${token}` }),
    ...options.headers,
  };

  const res = await fetch(`${API_BASE}${url}`, { ...options, headers });

  if (res.status === 401) {
    localStorage.removeItem('token');
    if (typeof window !== 'undefined') {
      window.location.href = '/login';
    }
    throw new Error('未登录或登录已过期');
  }

  const json = await res.json();
  if (!json.success) throw new Error(json.message || '请求失败');
  return json.data;
}

// 流式请求（AI 对话 SSE）
export async function apiFetchStream(url, body, onChunk) {
  const token = localStorage.getItem('token');
  const res = await fetch(`${API_BASE}${url}`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      ...(token && { Authorization: `Bearer ${token}` }),
    },
    body: JSON.stringify(body),
  });

  const reader = res.body.getReader();
  const decoder = new TextDecoder();

  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    const text = decoder.decode(value, { stream: true });
    onChunk(text);
  }
}
```

// hooks/useInfiniteMovies.js —— 无限滚动加载
function useInfiniteMovies(params = {}) {
  const [movies, setMovies] = useState([]);
  const [page, setPage] = useState(1);
  const [hasMore, setHasMore] = useState(true);
  const [loading, setLoading] = useState(false);

  const loadMore = useCallback(async () => {
    if (loading || !hasMore) return;
    setLoading(true);
    try {
      const data = await apiFetch(
        `/api/movies?page=${page}&pageSize=24&${new URLSearchParams(params)}`
      );
      setMovies(prev => [...prev, ...data.records]);
      setHasMore(data.page < data.totalPages);
      setPage(p => p + 1);
    } catch (e) {
      console.error(e);
    } finally {
      setLoading(false);
    }
  }, [page, loading, hasMore, params]);

  return { movies, loading, hasMore, loadMore };
}

### 8.2 代码质量要求

| 维度 | 标准 |
|------|------|
| **组件类型** | 纯展示用 Server Component；有交互/状态/事件用 'use client' |
| **样式方案** | 100% TailwindCSS，不自定义 CSS 文件 |
| **暗色主题** | 全站暗色 bg-[#0F0F0F]，不使用浅色模式 |
| **响应式** | sm(640) / md(1024) / lg(1280) / xl(1536) 四档 |
| **加载态** | 所有数据组件有骨架屏（Skeleton） |
| **错误态** | 每个页面有 error.js + 全局 Toast 错误提示 |
| **空状态** | 所有列表有空数据占位（Empty 组件） |
| **无限滚动** | 首页和列表页使用 IntersectionObserver 触发加载 |
| **Token** | JWT 存 localStorage，每次请求自动附加，401 跳登录页 |
| **图标** | Lucide Icons，24px，strokeWidth=1.5 |

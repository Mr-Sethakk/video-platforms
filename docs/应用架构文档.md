# 🎬 影视平台全栈应用架构文档

> **适用对象**：AI 代码生成工具（Claude Code / Cursor / Trae 等）
> **目标**：根据本文档一次性生成完整的全栈影视平台项目骨架
> **基于课程**：暑假实习「电影APP全栈开发 + AI智能体」30天教学大纲

---

## 一、项目概述

### 1.1 项目定位

构建一个面向C端用户的影视内容平台Web应用，支持电影浏览/搜索/收藏、AI智能客服对话、海报图片识别、AI视频/漫剧生成、内容审核上架等完整业务闭环。

### 1.2 核心功能模块

| 模块 | 功能 | 用户角色 |
|------|------|----------|
| 电影展示 | 电影列表、详情、搜索、分类筛选、评分排序 | 所有用户 |
| 用户系统 | 注册、登录、JWT认证、角色权限（USER/ADMIN） | 所有用户 |
| 片单收藏 | 个人收藏管理、收藏状态同步 | 登录用户 |
| AI聊天助手 | 自然语言电影推荐、问答、支持多轮对话和流式响应 | 所有用户 |
| 海报识别 | 上传电影海报图片 → OCR/多模态识别 → 返回电影信息 | 登录用户 |
| AI视频生成 | 文生视频（输入描述→生成视频）、任务状态轮询 | 登录用户 |
| AI漫剧制作 | 剧本→分镜→AI画面生成→TTS配音→视频合成 全流程 | 创作者 |
| 云雀管理后台 | 视频审核、内容管理、数据统计仪表盘 | 管理员 |
| 视频上架 | 上传→AI生成描述→审核→上架全流程 | 创作者/管理员 |

## 二、技术栈总览

### 2.1 推荐技术栈（生产级）

| 层级 | 技术 | 版本 | 说明 |
|------|------|------|------|
| **前端框架** | Next.js (App Router) | 14.x | React全栈框架，SSR/SSG/API Routes |
| **UI框架** | React | 18.x | 组件化开发 |
| **样式方案** | TailwindCSS | 3.4.x | 原子化CSS，AI友好 |
| **状态管理** | React Hooks (useState/useContext) | — | 轻量级，无需额外库 |
| **HTTP客户端** | fetch / axios | — | 前后端通信 |
| **图表库** | Recharts | 2.x | 管理后台统计图表 |
| **后端框架** | Spring Boot | 3.2.x | Java企业级后端 |
| **ORM** | MyBatis-Plus | 3.5.x | 增强版MyBatis，简化CRUD |
| **安全框架** | Spring Security + JWT | — | 认证授权 |
| **数据库** | MySQL | 8.0+ | 关系型数据库 |
| **缓存** | Redis | 7.x | 热点数据缓存、Session |
| **AI SDK** | Spring AI Alibaba | 1.0.0-M6 | 阿里云百炼/DashScope接入 |
| **AI模型** | 通义千问 (qwen-max) | — | 对话/推荐/问答 |
| **OCR/多模态** | DashScope 多模态API | — | 海报识别 |
| **视频生成** | 可灵AI API (或Runway/Pika) | — | AI视频生成 |
| **TTS** | 阿里云语音合成 | — | 漫剧配音 |
| **构建工具** | Maven (后端) / npm (前端) | — | 依赖管理和构建 |
| **JDK** | JDK 17+ | — | Java运行环境 |

### 2.2 备选轻量栈（Node.js路线）

若学习阶段不需要Java，可替换为：

| 层级 | 替换方案 |
|------|---------|
| 后端框架 | Express.js (Node.js) 替代 SpringBoot |
| ORM | Prisma 替代 MyBatis-Plus |
| AI接入 | 直接调用 DashScope HTTP API 替代 Spring AI Alibaba |

### 4.2 路由设计

| 路径 | 页面 | 渲染方式 | 说明 |
|------|------|----------|------|
| `/` | 首页 | SSR | Hero + 热门电影 + 分类入口 |
| `/movies` | 电影列表 | SSR | 搜索 + 筛选 + 分页 |
| `/movies/[id]` | 电影详情 | SSG | 静态生成 + ISR |
| `/search?q=xxx` | 搜索结果 | SSR | 关键词搜索 |
| `/watchlist` | 我的片单 | CSR (需登录) | 个人收藏 |
| `/upload` | 视频上传 | CSR (需登录) | 创作者上传 |
| `/admin` | 管理后台 | CSR (需管理员) | 数据统计 |
| `/admin/review` | 视频审核 | CSR (需管理员) | 审核面板 |
| `/login` | 登录 | CSR | 登录表单 |
| `/register` | 注册 | CSR | 注册表单 |

```sql
-- ============ 用户表 ============
CREATE TABLE `user` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `username` VARCHAR(50) UNIQUE NOT NULL COMMENT '用户名',
    `password` VARCHAR(255) NOT NULL COMMENT 'BCrypt加密密码',
    `email` VARCHAR(100) UNIQUE NOT NULL COMMENT '邮箱',
    `avatar_url` VARCHAR(500) COMMENT '头像URL',
    `role` ENUM('USER', 'ADMIN') DEFAULT 'USER' COMMENT '角色',
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    `updated_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
) COMMENT '用户表';

-- ============ 电影表 ============
CREATE TABLE `movie` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `title` VARCHAR(200) NOT NULL COMMENT '电影标题',
    `description` TEXT COMMENT '电影简介',
    `rating` DECIMAL(2,1) DEFAULT 0.0 COMMENT '评分(0-10)',
    `year` INT COMMENT '上映年份',
    `genre` VARCHAR(50) COMMENT '类型（剧情/科幻/爱情/动作等）',
    `director` VARCHAR(100) COMMENT '导演',
    `actors` VARCHAR(500) COMMENT '主演（逗号分隔）',
    `poster_url` VARCHAR(500) COMMENT '海报图片URL',
    `duration` INT COMMENT '时长（分钟）',
    `country` VARCHAR(50) COMMENT '国家/地区',
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    `updated_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    `deleted` TINYINT DEFAULT 0 COMMENT '逻辑删除',
    INDEX `idx_genre` (`genre`),
    INDEX `idx_rating` (`rating`),
    INDEX `idx_year` (`year`),
    FULLTEXT INDEX `ft_title_desc` (`title`, `description`)
) COMMENT '电影表';

-- ============ 片单收藏表 ============
CREATE TABLE `watchlist` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `user_id` BIGINT NOT NULL COMMENT '用户ID',
    `movie_id` BIGINT NOT NULL COMMENT '电影ID',
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE KEY `uk_user_movie` (`user_id`, `movie_id`),
    FOREIGN KEY (`user_id`) REFERENCES `user`(`id`) ON DELETE CASCADE,
    FOREIGN KEY (`movie_id`) REFERENCES `movie`(`id`) ON DELETE CASCADE
) COMMENT '片单收藏表';

-- ============ 用户评论表 ============
CREATE TABLE `comment` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `user_id` BIGINT NOT NULL,
    `movie_id` BIGINT NOT NULL,
    `content` TEXT NOT NULL COMMENT '评论内容',
    `rating` TINYINT COMMENT '评分(1-5)',
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (`user_id`) REFERENCES `user`(`id`) ON DELETE CASCADE,
    FOREIGN KEY (`movie_id`) REFERENCES `movie`(`id`) ON DELETE CASCADE
) COMMENT '用户评论表';

-- ============ 视频表 ============
CREATE TABLE `video` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `title` VARCHAR(200) NOT NULL COMMENT '视频标题',
    `description` TEXT COMMENT '视频描述',
    `url` VARCHAR(500) NOT NULL COMMENT '视频URL',
    `thumbnail` VARCHAR(500) COMMENT '缩略图URL',
    `duration` INT COMMENT '时长（秒）',
    `status` ENUM('PENDING', 'APPROVED', 'REJECTED') DEFAULT 'PENDING' COMMENT '审核状态',
    `user_id` BIGINT NOT NULL COMMENT '上传者ID',
    `category_id` BIGINT COMMENT '分类ID',
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    `updated_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (`user_id`) REFERENCES `user`(`id`),
    INDEX `idx_status` (`status`)

-- ============ 分类表 ============
CREATE TABLE `category` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `name` VARCHAR(50) UNIQUE NOT NULL COMMENT '分类名称',

    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,

) COMMENT '标签表';

-- ============ 视频-标签关联表 ============

    FOREIGN KEY (`video_id`) REFERENCES `video`(`id`) ON DELETE CASCADE,
    FOREIGN KEY (`tag_id`) REFERENCES `tag`(`id`) ON DELETE CASCADE
) COMMENT '视频标签关联表';
```

## 十一、安全设计

| 层面 | 措施 |
|------|------|
| 传输安全 | HTTPS（生产环境）。开发环境使用 HTTP |
| 认证 | JWT Token，24小时过期。登录返回access_token |
| 密码存储 | BCrypt 加密（强度10），永不明文存储 |
| 接口授权 | SpringSecurity + @PreAuthorize 方法级权限控制 |
| 角色控制 | USER（浏览+收藏+上传）/ ADMIN（审核+管理） |
| CORS | 仅允许前端域名跨域 |
| SQL注入 | MyBatis-Plus 参数化查询，天然防注入 |
| XSS | React 默认转义输出 + CSP Header |
| 文件上传 | 校验文件类型（白名单）+ 大小限制 + 病毒扫描（生产） |
| 敏感信息 | API Key/数据库密码 通过环境变量注入，不写入代码 |
